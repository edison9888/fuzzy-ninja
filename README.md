#设计思路：

##网页端：

chrome插件，自动在当前网页中嵌入与sever通讯的相关基于jquery的js代码，打开时生成二维码，供连接用。

##通讯服务器：

基于node.js和faye的sever，在手机和网页端之间接收并转发指令。

##手机端：

iOS应用，设计参考了ipod的转轮设计，使用时先扫瞄chrome插件中的二维码并进行连接。连接之后可以对网页中的播放器进行操作。
